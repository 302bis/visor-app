BD Analyst Sr.

Si estás acostumbrado al mundo de la alta empresa mexicana, y tienes la capacidad de relacionarte de forma excepcional con directores y gerentes de nivel interacional, te estamos buscando.









Puesto:
Director de Ventas
Área Laboral
Ventas
Expectativas

El desafío relacionado con esta posición es el de apoyar a los Directores de business development en las actividades comerciales que la empresa realiza con multinacionales, instituciones financieras y otras socios de negocio (Enterprise Sales). Como parte integral del equipo comercial de la empresa la misión es la de generar una base de usuarios para la plataforma de financiamientos. La actividades de este rol incluyen: negociación, documentación de la relación comercial (SOW´s), diseño de cronogramas, análisis de oportunidades con clientes y seguimiento del las ventas en forma de gestión de proyectos.

Actividades y Responsabilidades
  • Dominar y ofrecer los productos y servicios de Visor. ( A1 )
  • Ingresar y actualizar diariamente la información de los Leads en CRM (Zoho) ( A3 )
  • Agendar y hacer Follow Up de los Leads. ( A4 )
  • Crear la oferta de valor (.ppt o SOW del Cliente). ( A5 )
  • Documentar minutas (24 hrs. máximo después de la junta). ( A6 )
  • Identificar nuevas oportunidades  de venta ( A7 )
  • Diseñar soluciones con el uso de la plataforma de Visor para clientes y prospectos. ( A8 )
  • Establecer mensualmente Steering Committees / juntas de avance. ( A10 )
  • Solicitar paquete de información básica al prospecto (NDA, estados financieros, presentación corporativa, paquete legal) - Checklist de Información Básica. ( A11 )
  • Elaborar Business Case de Banco. ( A12 )
  • Crear Business Case de Large Company y diseño de soluciones con el uso de la plataforma de Visor. ( A13 )
  • Enviar Term Sheet. ( B3 )
  • Definir necesidades de campañas / requerimientos  / parámetros para la configuración de la plataforma. ( B5 )
  • Lanzamiento de iniciativa (equipo Visor). ( B6 )
  • Elaborar SOW. ( C1 )
  • Identificar nuevas experiencias a clientes (Large Companies). ( C12 )
  • Obtener las firmas en el SOW en función de los avances de proyecto (por versión). ( C20 )
  • Definir Cronograma de Proyecto. ( C21 )
  • Realizar y garantizar el cumplimiento del SOW (Statements of Work). ( C23 )
  • Formalizar contrato de prestación de servicios. ( C26 )
  • Adaptar los Anexos del contrato.  ( C27 )
  • Coordinar reuniones con cliente. ( C28 )
  • Mantener estricta confidencialidad sobre la información de prospectos, instituciones financieras y clientes. ( C29 )
  • Generar el análisis de impacto de las campañas a considerar. ( D2 )
  • Crear la propuesta de valor  hacia el cliente (impacto financiero). ( D3 )
  • Realizar la segmentación de portafolio para realizar campañas telefónicas asi como electrónicas (via mails, recordatorios, SMS y otros). ( E1 )
  • Solicitar archivo recurrente.  ( G3 )
  • Dar soporte al equipo de Account Management para garantizar la conversión de las PyMEs de la campaña. ( G11 )
  • Estructurar planes de contingencia en caso de que ciertos prospectos no avancen conforme al Plan.   ( G12 )
  • Analizar la utilización de la línea de crédito a partir del objeto del crédito y en su caso redefinir destino. ( L1 )
  • Dar seguimiento permanente a cada uno de los créditos colocados de su cartera, allegándose de toda aquella información relevante que indique la situación de los créditos en cuestión. ( L4 )
Habilidades y Competencias

Habilidades: 

  • Atención a detalle.
  • Capacidad analítica /abstracción.
  • Entendimiento de las necesidades cambiantes de clientes.
  • Automotivado.
  • Comunicación efectiva con clientes / otras áreas.
  • Enfoque a resultados.  
  • Aprecio por soluciones tecnológicas a problemas de negocio y uso de plataformas.

 

Competencias: 

  • Alta responsabilidad.
  • Proactividad.
  • Capacidad analítica para encontrar soluciones simples a problemas complejos.
  • Alta capacidad de respuesta a retos constantes. 
  • Enfoque a resultados.
  • Conocimiento de bases html o css, saber agregar valores. 
  • Uso de herramientas de Office (Project / Visio avanzados) y conocimiento de plataformas web.
  • Cierre de ventas apalancado en metodología de gestión de proyectos.
  • Trabajo efectivo bajo presión. 
  • Tolerancia a la frustración.

 

Idioma: Inglés avanzado (fluido).
